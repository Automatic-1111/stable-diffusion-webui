# Stable Diffusion WebUI (A1111)

The most popular and powerful web UI for Stable Diffusion.

## Key Features
- txt2img / img2img / inpainting / outpainting
- ControlNet, IP-Adapter, LoRA, LyCORIS
- SDXL, SD 3.5, Flux.1, Pony, AuraFlow
- Built-in training & extensions
- Works on NVIDIA, AMD, Intel, Apple Silicon

## Quick Start

**Windows** → run `webui-user.bat`  
**Linux/macOS** → run `./webui.sh`

Opens at http://127.0.0.1:7860  
First launch downloads everything automatically.

Enjoy!
